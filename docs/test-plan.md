# EV充电平台 - 全面测试计划书

## 测试概述

对EV充电平台进行三维度全面测试：**拟人化测试**、**恶劣环境测试**、**API安全测试**。

### 测试范围
- 后端API: Gateway(8080) + 5个业务服务
- 前端应用: admin-web(5173) + simulator-web(5177)
- 基础设施: PostgreSQL + Redis + Kafka

### 发现的问题
1. **Gateway崩溃风险**: 发送畸形JSON/超长输入会导致Gateway JVM崩溃 (严重)
2. **Dashboard toFixed错误**: 订单电量字段为null时前端报错 (轻微)
3. **OPS App数据**: 部分使用Mock数据而非真实后端 (中等)
