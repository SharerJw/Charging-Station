# ⚡ 充电平台 - 用户端（小程序）深度细化提示词

针对**用户端（小程序）**，我将其拆分为 **9个核心模块** 进行深度细化。涵盖找站导航、扫码充电全流程、订单支付、钱包充值、车辆管理、会员体系、评价社区、消息通知、个人设置等完整用户旅程。

---

## 提示词 1：全局架构、授权登录与首页（地图找站）

> **定位**：小程序的技术架构、微信授权流程、以及最核心的首页地图找站体验。

```text
请生成充电平台用户端微信小程序的全局架构、授权登录流程和首页（地图找站）页面。
技术栈：UniApp + Vue 3 (Composition API) + TypeScript + Pinia + uni-ui。
目标平台：微信小程序（同时兼容支付宝/抖音小程序），基础库版本 >= 2.25.0。
UI风格：清新活力、亲和力强，主色调 充电绿(#07C160)，辅助色 活力蓝(#1677FF)，背景色 #F6F7FB，大圆角(16rpx)，卡片轻投影。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【全局架构设计】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 底部 TabBar（4个Tab，自定义样式）：
   - 🏠 首页（地图找站）
   - 📋 订单（充电记录）
   - 💰 钱包（余额/优惠券）
   - 👤 我的（个人中心）
   - Tab 图标：未选中灰色线框图标，选中绿色实心图标。
   - 订单 Tab 有进行中订单时显示绿色角标"进行中"文字。
   - TabBar 上方浮层：当有正在进行的充电订单时，显示"充电中"悬浮条（实时显示充电时长+费用，点击跳转充电详情页）。

2. 全局顶部导航栏（自定义导航栏组件）：
   - 左侧：定位图标 + 当前城市名（如"北京市"，点击弹出城市选择器）。
   - 中间：搜索栏（占位符"搜索充电站/地址"，点击进入搜索页）。
   - 右侧：扫一扫图标（核心入口，大尺寸，带呼吸灯效果）+ 消息铃铛（带红点）。

3. 全局悬浮按钮（右下角）：
   - [📞 客服] 圆形悬浮按钮，点击弹出客服选项（在线客服/电话客服）。
   - 充电中状态时变为 [⚡ 充电中] 按钮，点击跳转充电详情。

4. 全局组件：
   - 网络异常弹窗：断网时显示"网络不太给力" + [重试] 按钮 + 可爱插画。
   - 定位授权引导：未授权定位时的全屏引导页，说明为什么需要定位 + [去授权] 按钮。
   - 更新提示弹窗：有新版本时提示更新（小程序强制更新场景）。
   - 全局 Loading：品牌 Logo 旋转 + "加载中..." 文字。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【授权登录流程】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 启动页（首次进入/冷启动）：
   - 品牌 Logo 动画（充电插头+闪电动效，持续 1.5 秒）。
   - 底部 Slogan："随时随地，满电出发 ⚡"。

2. 引导页（仅首次安装显示，3页滑动）：
   - Page 1：插画 + "附近找站，一键导航" + 跳过按钮。
   - Page 2：插画 + "扫码充电，即扫即充"。
   - Page 3：插画 + "智能推荐，省钱充电" + [立即体验] 按钮。

3. 登录方式（非强制登录，浏览功能免登录）：
   - 触发登录时机：扫码充电、下单支付、查看订单、修改个人信息时触发。
   - 登录弹窗（半屏弹窗，底部弹出）：
     * 品牌 Logo + "登录后享受完整充电服务"。
     * [微信一键登录] 大按钮（绿色，调用 wx.getUserProfile 获取头像昵称，wx.login 获取 openid）。
     * [手机号快捷登录] 按钮（使用微信手机号快速验证组件，一键获取手机号）。
     * [手机号+验证码登录] 文字链接（传统方式，输入手机号→获取验证码→登录）。
     * 底部：用户协议 + 隐私政策勾选框（必勾才能登录，未勾选时按钮抖动提示）。
   - 登录成功后：Toast "登录成功 🎉" + 新用户自动发放欢迎礼包提示。

4. 静默登录策略：
   - 打开小程序时静默调用 wx.login 获取 openid，后端自动创建/关联用户。
   - 仅展示头像昵称等个人信息时才弹出授权窗口。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【首页 - 地图找站（核心页面）】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

页面结构（自上而下）：

1. 地图区域（占据页面上方 55% 高度）：
   - 腾讯地图（小程序原生 map 组件），默认显示用户当前位置。
   - 定位蓝色光点 + 精度圈。
   - 充电站标注点（自定义 Marker 图标）：
     * 空闲状态：绿色充电插头图标 + 空闲枪数气泡（如"空3"）。
     * 繁忙状态：橙色图标 + 空闲枪数（如"空1"）。
     * 已满状态：红色图标 + "满"标签。
     * 离线/不可用：灰色图标 + "暂停"标签。
     * 超充站标识：紫色闪电图标 + "超充"标签。
   - Marker 聚合策略：
     * 缩放级别 ≤12：显示聚合气泡（如"朝阳区 128站"），颜色深浅表示密度。
     * 缩放级别 13-15：显示单独站点 Marker。
     * 缩放级别 ≥16：显示站点范围内每个充电桩的位置。
   - 地图交互：
     * 双指缩放、单指平移。
     * 右下角：[回到当前位置] 按钮（定位图标，点击后地图回到用户位置 + 指南针朝向修正）。
     * 右下角：[图层切换] 按钮（标准地图 / 卫星地图 / 路况地图）。
     * 长按地图：在该位置搜索附近站点。
   - 筛选浮层（地图顶部，半透明胶囊按钮组）：
     * [全部] [快充] [慢充] [超充]（充电类型筛选）。
     * [空闲] 开关（仅显示有空闲枪的站点）。
     * [24h] 开关（仅显示24小时运营站点）。
     * [免费停车] 开关。
     * [⚙ 更多筛选] → 弹出底部筛选面板。
   - 更多筛选面板（底部弹出半屏）：
     * 价格区间：滑块选择（¥0.5~¥2.5/度）。
     * 距离范围：1km / 3km / 5km / 10km / 不限。
     * 配套设施：多选标签（卫生间/便利店/停车场/休息室/洗车/餐饮/WiFi）。
     * 站点类型：公共站 / 专用站 / 目的地站。
     * 排序方式：距离最近 / 价格最低 / 评分最高 / 空闲最多。
     * [重置] [确认] 按钮。

2. 站点底部列表（占据页面下方 45%，可上下拖拽调整比例）：
   - 顶部把手区域（拖拽指示条）：
     * 向上拖拽扩大列表区域（最大 85%），向下拖拽缩小（最小 30%）。
     * 显示当前列表条目数："附近 32 个充电站"。
   
   - 推荐/排序切换：
     * 横向滚动胶囊：🎯 智能推荐 | 📍 距离最近 | 💰 价格最低 | ⭐ 评分最高 | ⚡ 充电最快
   
   - 站点卡片列表（scroll-view，下拉刷新 + 上拉加载）：
     * 每张站点卡片设计（白底圆角卡片，高度约 200rpx）：
       
       左侧（25%）：
       - 站点封面缩略图（圆角，72×72rpx）。
       - 距离标签（如"580m"，绿色文字）。
       
       中间（55%）：
       - 第一行：站点名称（加粗，16号字）+ 评分（⭐4.8，黄色）。
       - 第二行：地址（灰色小字，单行截断）。
       - 第三行：标签组（横向排列小标签）：
         * 充电类型标签："快充"蓝底 / "慢充"绿底 / "超充"紫底。
         * 特色标签："24h" / "免费停车2h" / "卫生间"（灰色描边标签）。
       - 第四行：
         * 价格信息："¥1.28/度"（含电费+服务费，绿色大字号）。
         * 空闲状态："空闲 4/8"（绿色文字）或"排队中"（红色文字）。
       
       右侧（20%）：
       - 导航图标按钮（蓝色圆形，点击调起地图导航）。
       - [详情] 文字按钮。
     
     * 卡片交互：
       - 点击卡片 → 进入站点详情页。
       - 地图 Marker 点击时，列表自动滚动到对应站点卡片并高亮闪烁。
       - 列表滚动时，地图自动居中到列表中间位置站点的坐标（联动效果）。

3. 搜索功能（点击顶部搜索栏进入搜索页）：
   - 搜索输入框（自动聚焦，带清除按钮）。
   - 搜索历史（可清空）。
   - 热门搜索词（如 "特斯拉超充"、"24小时充电站"）。
   - 搜索结果列表：
     * 站点结果（带距离、评分、价格）。
     * 地址/POI 结果（点击定位到该位置并搜索附近站点）。
   - 搜索联想（输入时实时推荐匹配的站点和地址）。

要求：
- 地图首次加载使用骨架屏占位（灰色地图轮廓 + 闪烁标注点占位）。
- 地图 Marker 使用自定义 callout（自定义气泡弹窗），非默认 callout。
- 站点列表使用长列表性能优化（ recycle-list / virtual-list 方案）。
- 定位失败时的降级方案：显示城市中心 + 手动选择位置引导。
- 地图和列表的联动需流畅（使用 IntersectionObserver 或 scroll-into-view）。
- 首屏加载 < 1.5秒（分包加载，地图 SDK 按需加载）。
- 小程序冷启动时并行请求：定位 + 附近站点 + 用户信息。
```

## 提示词 2：站点详情与充电桩选择

> **定位**：用户点击站点后看到的完整站点信息、可用充电桩列表、预约功能。

```text
请生成充电平台用户端小程序的"站点详情页"和"充电桩选择"页面。
技术栈：UniApp + Vue 3 + Pinia + 腾讯地图 SDK。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【页面一：站点详情页】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

页面采用滚动长页面设计（scroll-view），顶部地图 + 下方信息流。

1. 顶部地图区（高度 400rpx，可点击放大全屏）：
   - 地图显示站点位置（红色 Marker）+ 用户当前位置（蓝色）+ 导航路线预览。
   - 左上角：返回按钮（半透明圆形背景）。
   - 右上角：[分享] 按钮 + [全屏地图] 按钮。
   - 地图底部渐变遮罩上叠加：站点名称（大字白色）+ 距离。

2. 站点基本信息卡片：
   - 站点名称（20号字加粗）+ 评分星星 + 评价数（如"4.8分 · 326条评价"）。
   - 站点类型标签组："公共快充站" / "24小时运营" / "免费停车"。
   - 地址行：详细地址文字 + [📋复制] + [🗺导航] 按钮。
   - 营业时间：如"全天24小时" 或 "06:00-23:00"（非营业时间显示红色"当前已关闭"）。
   - 联系电话：电话号码 + [📞一键拨打] 按钮。

3. 实时可用充电桩列表（核心区域）：
   - 顶部汇总条：
     * 总桩数 XX 台 | 空闲 X | 充电中 X | 故障 X。
     * 充电类型筛选胶囊：全部 | 直流快充 | 交流慢充 | 液冷超充。
   
   - 充电桩卡片列表（每台桩一张卡片）：
     * 卡片左侧：
       - 充电桩编号（如"#3号桩"）+ 型号标签（如"120kW 直流快充"）。
       - 充电枪列表（每把枪一行）：
         · 枪号（如"A枪"）+ 状态指示灯（🟢空闲 / 🔵充电中 / 🔴故障）。
         · 空闲时显示：[立即充电] 绿色按钮。
         · 充电中时显示：预计剩余时间 + 当前SOC（如"预计25min · 当前68%"）。
         · 故障时显示：故障原因摘要 + "暂不可用"灰色文字。
     * 卡片右侧：
       - 当前电价信息：
         · 电费：¥X.XX/度。
         · 服务费：¥X.XX/度。
         · 合计：¥X.XX/度（绿色大字）。
         · 时段提示："当前为平段 · 14:00 起切换峰段 ¥X.XX/度"。
   
   - 充电桩详情弹窗（点击充电桩卡片展开）：
     * 设备型号、最大功率、输出电压范围。
     * 支持的充电接口类型（国标2015/特斯拉/其他）。
     * 近24小时使用率时间轴（24格，颜色深浅表示忙闲程度）。
     * [预约此桩] 按钮。

4. 电价详情卡片：
   - 24小时电价时间轴可视化：
     * 横轴：0:00 ~ 24:00。
     * 纵轴：综合单价（元/度）。
     * 不同时段用色块标注：尖峰(红) / 峰(橙) / 平(蓝) / 谷(绿) / 深谷(深绿)。
     * 当前时间点用竖线标注 + 当前价格气泡。
     * 支持手指左右滑动查看各时段详细价格。
   - 电价明细表格（可折叠展开）：
     * 各时段的电费、服务费、合计单价。
   - 省钱提示："💡 建议 23:00-07:00 充电，电费低至 ¥0.35/度，可省 ¥XX"。

5. 站点图片相册：
   - 图片横向滚动（大图预览，宽度 320rpx × 高度 240rpx）。
   - 图片分类标签：全部 / 外观 / 设备 / 环境 / 配套。
   - 点击图片进入全屏画廊模式（支持左右滑动、双指缩放）。
   - [📷 上传图片] 按钮（用户UGC贡献，上传获积分奖励）。
   - VR全景入口（如有）：[🌐 查看VR全景] 按钮。

6. 配套设施与周边信息：
   - 设施图标网格（2×4）：
     * 🅿停车场 | 🚻卫生间 | 🏪便利店 | ☕休息室
     * 🚿洗车 | 🍜餐饮 | 📶WiFi | 🔧自助工具
     * 有的设施高亮绿色，没有的灰色。
   - 停车费说明："充电免费停车2小时，超时 ¥5/小时"。
   - 周边 POI 推荐（附近餐饮、商场、咖啡厅，来自地图数据）。

7. 用户评价区域：
   - 综合评分展示：大数字"4.8" + 评分分布条形图 + 评价总数。
   - 热门评价标签云："充电速度快(86)" / "环境整洁(54)" / "价格实惠(43)" / "排队严重(12)"。
   - 最新评价列表（显示3条，[查看更多]跳转评价列表页）：
     * 每条：用户头像+昵称 + 评分星星 + 评价内容(2行截断) + 充电时间 + 照片缩略图。
   - [✍ 写评价] 按钮（如用户在此站点有过充电订单）。

8. 底部固定操作栏（安全区适配）：
   - 左侧：[🗺 导航前往] 按钮（蓝色描边，点击调起腾讯/高德/百度地图）。
   - 右侧：[⚡ 扫码充电] 大按钮（绿色渐变，占据 60% 宽度，核心CTA）。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【页面二：预约充电】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 预约入口：
   - 从站点详情页 [预约此桩] 进入。
   - 或在首页筛选中勾选"可预约"站点。

2. 预约表单页：
   - 选择充电桩：从可用桩列表中选择。
   - 选择预约时间段（日历+时间选择器）：
     * 最早可预约：30分钟后。
     * 最晚可预约：7天内。
     * 时间粒度：15分钟间隔。
   - 预计充电量/时长（可选填，帮助系统优化排程）。
   - 预约费用说明：
     * "预约免费" 或 "预约费 ¥X（到达后抵扣充电费）"。
```
