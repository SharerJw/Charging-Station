# 🖥️ 充电平台 - 后台管理系统（Web）深度细化提示词

针对**后台管理系统（Web）**，我将其拆分为 **10个核心模块** 进行深度细化。涵盖运营驾驶舱、站点设备管理、订单财务、用户营销、电价策略、数据分析、权限安全等全维度。

## 提示词 2：运营数据驾驶舱（大屏 + 实时监控）

> **定位**：面向管理层的全局运营数据可视化大屏，以及面向运维团队的实时监控中心。

```text
请生成充电平台后台管理的"运营数据驾驶舱"模块，包含运营大屏和实时监控中心两个子页面。
技术栈：Vue 3 + ECharts 5 + DataV + WebSocket + 高德地图 JS API。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【子页面一：运营数据大屏（适配 1920×1080 / 3840×2160 大屏）】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

整体风格：深色科技风（深蓝渐变背景 #0A1628→#1A2744），霓虹色数据高亮，发光边框效果。
支持全屏投放模式（F11 或专用全屏按钮，隐藏所有管理界面元素）。

【顶部信息条（高度 80px）】

【中央核心区（宽度 40%）】
1. 上部 - 核心指标组（3个发光数字面板）：
   - 累计充电量（MWh）：大数字翻牌器动效（如 9 位数翻牌）+ "今日 +XX,XXX kWh"。
   - 累计服务用户数：翻牌器 + "今日新增 +XXX"。
   - 累计营收（万元）：翻牌器 + "今日 +¥XX,XXX"。

2. 中部 - 全国站点分布地图（占据核心区 70% 面积）：
   - 3D 中国地图（ECharts GL 或高德地图 3D 视图），带省界轮廓发光效果。

     * 聚集点（多个站点密集区域）：聚合气泡显示总数。
     * 脉冲动画：有实时充电订单的站点显示脉冲波纹扩散效果。
   - 交互：
     * 鼠标悬停气泡弹出站点摘要卡片（站名/设备数/在线率/今日营收）。
     * 点击气泡下钻到省级地图→城市地图→具体站点列表。
     * 自动轮播：每 15 秒自动聚焦一个重点城市，展示该城市详细数据。
   - 飞线动画：从各城市向总部城市发射光线粒子（象征数据汇聚）。

3. 下部 - 24小时充电趋势（面积图）：
   - X轴：24小时时间轴（标注峰谷平时段色块）。

   - 双层面积图：上层=今日实时数据（蓝色实线+渐变填充），下层=昨日同期（灰色虚线）用于对比。

1. 左上 - 站点利用率排行 Top10：

   - 中心显示设备总数。

3. 左下 - 今日告警统计：
   - 上部：告警级别分布（3个发光数字：紧急X / 一般X / 提醒X）。
   - 下部：最近 5 条告警滚动列表（新告警从底部滑入，带闪烁高亮效果）。

【右翼区域（宽度 30%），3个垂直堆叠面板】
1. 右上 - 营收构成分析：
   - 嵌套饼图：内圈=电费收入/服务费收入/停车费收入 占比。
   - 外圈=各支付方式占比（微信/支付宝/银联/余额/企业账户）。

3. 右下 - 实时订单流：

     * 每条显示：用户昵称(脱敏) + 站点名 + 充电量 + 金额 + 时间。
     * 大额订单（>100元）金色高亮。

【底部信息条（高度 40px）】
- 左侧跑马灯：运营公告/安全提示滚动文字。
- 右侧：系统运行状态（服务器CPU/内存/网络IO指标）。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 顶部筛选栏：
```

## 提示词 4：设备台账与远程控制管理

> **定位**：充电桩设备的全生命周期管理（入库→安装→运营→维护→报废），以及批量远程操控能力。

```text
请生成充电平台后台管理的"设备台账与远程控制管理"模块。
技术栈：Vue 3 + Element Plus + Monaco Editor + ECharts + WebSocket。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   - 设备总数：X,XXX 台（较上月 +XX）
   - 在线设备：X,XXX 台（在线率 XX.X%）
   - 故障设备：XX 台（故障率 X.X%）
   - 今日充电设备：X,XXX 台（充电中 X / 空闲 X）

2. 筛选区：
   - 搜索框：设备编号/序列号/SIM卡号。
   - 站点选择（下拉 + 树形）。
   - 设备类型：交流慢充 / 直流快充 / 液冷超充 / 便携式。
   - 设备型号（动态加载：如 "星星充电-120kW-V3"、"盛弘-240kW"等）。
   - 通信协议：OCPP 1.6J / OCPP 2.0.1 / 私有协议。
   - 设备状态：在线 / 离线 / 故障 / 维护中 / 已报废。
   - 固件版本（下拉选择，便于筛选需要升级的设备）。
   - 安装时间范围。

3. 数据表格（虚拟滚动）：
   - 列：
     * 设备编号（可复制，点击进入详情）
     * 设备型号 + 缩略图小图
     * 所属站点
     * 通信协议 + 固件版本
     * 设备状态（彩色状态点 + 文字）
     * 充电状态（空闲/充电中/故障）
     * 充电枪数
     * 今日充电量(kWh)
     * 累计充电量(MWh)
     * SIM卡ICCID
     * 最后心跳时间
     * 安装日期
     * 操作：远程控制 | 编辑 | 维修记录 | 固件升级 | 报废
   - 表格行颜色：故障设备行浅红背景、离线设备行浅灰背景。
   - 展开行详情（点击行展开）：
     * 实时电气参数（电压/电流/功率/温度）。
     * 最近 5 条告警摘要。
     * 近 7 天充电量迷你折线图。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【页面二：设备详情面板（抽屉/独立页面）】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 设备画像区：

     * 通信协议、固件版本、硬件版本。
     * SIM 卡号(ICCID)、IP 地址。
     * 额定功率、充电枪数、输入电压范围、输出电压范围。

     * A相电压/B相电压/C相电压。
     * 输出电流。
     * 实时功率 / 额定功率百分比。
     * 机内温度 / 枪头温度。

3. 远程控制操作台（分安全等级）：

     * [获取设备状态] → 发送 GetConfiguration 并展示返回结果。

     * [拉取设备日志] → 远程下载设备运行日志文件。

     * [远程重启]：选择重启类型(Soft/Hard) → 下发 Reset 指令 → 监控重启过程。

     * [下发配置参数]：可视化配置表单，修改后下发 ChangeConfiguration。
   
   - 等级3 - 高危类（需审批流）：
     * [远程固件升级]：选择目标版本 → 填写升级原因 → 提交审批 → 审批通过后执行。
     * [功率降额/调度]：调整最大功率输出。
     * [恢复出厂设置]：三级确认 + 审批。

4. 历史数据 Tab：
   - 充电记录：该设备的所有充电订单列表。
   - 告警记录：该设备的所有告警列表（支持时间范围筛选）。
   - 维修记录：历次维修/更换备件记录。
   - 固件升级记录：历次OTA升级记录（版本变更时间线）。
   - 通信日志：原始 WebSocket 报文记录（可筛选、可导出）。

5. 数据分析 Tab：
   - 充电量趋势（近30天日充电量柱状图）。
   - 利用率趋势（近30天日利用率折线图）。
   - 故障频次统计（近6个月故障次数趋势 + 故障类型分布饼图）。
   - 能耗效率分析（输入电量 vs 输出电量，计算转换效率趋势）。

【页面三：批量远程操控中心】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 目标设备选择：
   - 按条件筛选：站点/型号/固件版本/区域。
   - 手动添加：从设备列表中勾选添加。
```
